Display SpaceX's Successful and Failed Launches

To update the chart follow these steps!
  1. Select the checkbox options to choose whether you'd like to see successful or failed SpaceX missions or both!
  2. Press the create chart button to create the instance of the chart you'd like to see!
  3. If you wish to update the chart with different checkbox options, simply change the checkboxes and click the the create chart button again.
  4. If you wish to delete the chart all together you can click the delete chart button!
